# Website Architecture and Copy Outline

## Positioning
This site is designed as a lightweight academic personal website for:
- professional identity
- research visibility
- publication showcase
- teaching presentation
- collaboration contact

## Homepage structure
1. Hero
   - Name
   - Title and affiliation
   - 2–3 sentence research introduction
   - Buttons: publications, CV
   - Quick links: email, Google Scholar, ORCID, GitHub

2. About
   - Short biography
   - Current research focus
   - Methods and academic orientation

3. Research
   - 3 main research themes
   - Each theme described in 1–2 sentences

4. Publications
   - Selected publications or working papers
   - Each item includes title, status, and short description

5. Teaching
   - Courses
   - Teaching focus

6. Projects
   - Current projects
   - Brief descriptions

7. Contact
   - Email
   - Affiliation
   - Academic profile links

## Copy strategy
- Tone: professional, clear, internationally legible
- Style: concise but credible
- Audience: journal editors, collaborators, students, reviewers, grant committees

## What should be customized first
1. Email
2. Google Scholar link
3. ORCID
4. GitHub or LinkedIn
5. Real publication list
6. Real photo
7. Real CV PDF
8. Canonical domain and sitemap URL

## Recommended future expansion
- `/publications/` full publication page
- `/projects/` research projects page
- `/teaching/` teaching portfolio page
- `/news/` invited talks, awards, media, workshops
- `/data/` code and datasets
